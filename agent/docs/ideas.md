# Product Ideas

## Lazy Analyst (2026-01-18)

**Идея:** Не вызывать Analyst автоматически, а дать пользователю выбор.

**Flow:**
1. Вопрос → Barb + QueryBuilder + DataFetcher (~800ms)
2. UI показывает: "Данные готовы" + таблица/график
3. Кнопки: `[Проанализировать]` `[Сам разберусь]`
4. Если нажал "Проанализировать" → Analyst (+3-5s)

**Плюсы:**
- Мгновенный фидбек (<1s вместо 5-6s)
- Экономия токенов если юзер сам хочет смотреть
- Трейдеры часто хотят просто цифры

**Автоматика:**
- Мало данных (1-3 rows) → сразу с анализом
- Много данных (50+ rows) → предложить выбор

**Статус:** На подумать, не приоритет.
